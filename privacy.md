# Politique de confidentialité

**Dernière mise à jour : avril 2025**

## 1. Données collectées
Cet agent traite uniquement les documents uploadés volontairement 
par l'utilisateur (polices d'assurance).

## 2. Utilisation des données
Les documents sont analysés uniquement pour générer un résumé. 
Aucune donnée n'est conservée après la session.

## 3. Partage des données
Aucune donnée personnelle n'est partagée avec des tiers.
